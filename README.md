# Pico & Placa
Basic JS predictor to check if your vehicle can circulate with the specified plate number, date and time.

## Usage

You can run the usage example code in index.js in your terminal with node.
```js
$ node index.js
```

Simply replace the following three values in the example code (index.js) in the corresponding format:
```js
// User Inputs
const plate = "afk-123";
const date = "2021/03/9";
const time = "17:05";
```

## Run test

To run test simply:

```
$ yarn install
$ yarn test
```
